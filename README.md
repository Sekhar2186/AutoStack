# ✨ Overview

AutoStack is an AI-powered full-stack application generator that converts natural language prompts into complete production-ready Next.js applications.

Instead of generating only UI components, AutoStack plans, designs, validates, fixes, previews and exports complete applications.

It includes intelligent AI agents, live WebContainer preview, downloadable source code, PDF documentation generation, automatic validation and multi-provider AI support.

---

# 🎯 Features

## 🤖 AI Powered Generation

- Prompt → Full Application
- Intelligent Project Planning
- Automatic Route Generation
- Component Generation
- Page Generation
- Documentation Generation
- Architecture Generation

---

## 🧠 Multi-Agent Architecture

AutoStack uses specialized AI agents instead of a single prompt.

| Agent | Responsibility |
|--------|---------------|
| 🧠 Planner Agent | Creates application blueprint |
| 🧩 Component Agent | Generates reusable UI components |
| 📄 Page Agent | Generates pages |
| 🛣 Route Agent | Creates routing structure |
| 📚 Documentation Agent | Generates README & project docs |
| 🔧 Fix Agent | Repairs generated code automatically |
| ✅ Validator | Validates generated code before final output |

---

# ⚡ Intelligent Validation

Before any project is finalized AutoStack automatically validates:

- ✅ Next.js App Router Rules
- ✅ Client / Server Components
- ✅ Nested Link Detection
- ✅ Missing Imports
- ✅ Invalid HTML
- ✅ Layout Validation
- ✅ Duplicate Components
- ✅ Common React Errors

If issues are detected they are automatically fixed by the AI Fix Agent.
